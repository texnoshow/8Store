8Store (beta) 🛍️
8Store — это легкий и современный клиент-магазин приложений для Windows 8.1 и Windows 10, вдохновленный классическим дизайном Metro UI. Проект создан для удобного поиска и загрузки приложений в формате .appx.

✨ Особенности
Гибридный интерфейс: Поддержка системной панели Charms Bar в Windows 8.1 и встроенной кнопки настроек для Windows 10.

Умная настройка: Возможность вручную выбрать любую папку для загрузок через стандартный проводник (FolderPicker).

Адаптивность: Автоматическое определение версии Windows для отображения соответствующих элементов управления.

Простой каталог: Загрузка списка приложений в реальном времени из JSON-файла на GitHub.

🚀 Технологии
Язык: VB.NET

Платформа: WinRT (Windows Store App)

Среда разработки: Visual Studio 2013 / 2015

Формат данных: JSON (DataContractJsonSerializer)

🛠️ Как это работает
Приложение при запуске проверяет, выбрана ли папка для сохранения файлов. Если нет — оно предложит её выбрать. На Windows 10 в верхней части экрана всегда доступна кнопка «Настройки», вызывающая панель управления (Settings Flyout), где можно в любой момент изменить путь загрузки.

📥 Как установить (Инструкция)
Шаг 1: Используй MetroUnlocker (Win8.1)

Шаг 2: Установка сертификата
Чтобы Windows разрешила установку, нужно «довериться» сертификату приложения:

Нажмите правой кнопкой мыши на файл .appx.

Выберите Свойства -> вкладка Цифровые подписи.

Выберите подпись в списке и нажмите Сведения -> Просмотр сертификата.

Нажмите Установить сертификат.

Выберите Локальный компьютер.

Поместите сертификат в хранилище: Доверенные корневые центры сертификации (Trusted Root Certification Authorities).

Шаг 3: Запуск пакета
После установки сертификата просто дважды кликните по файлу .appx и нажмите кнопку «Установить» (Install) во всплывающем окне Windows (10) или add-appxpackage (Win8.1).

🛠 О приложении
8Store позволяет скачивать приложения из стороннего репозитория.

Настройка пути: При первом запуске нажмите кнопку «Настройки» (или используйте панель Charms Win+I на Windows 8.1), чтобы указать папку, куда будут сохраняться .appx файлы.

Работа с файлами: После скачивания вы найдете установочный файл приложения в выбранной вами папке.
